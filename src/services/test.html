More details

As you can see for the formatting of this document, only very basic things are available to you. This is for you to set up your content for when time comes for us to import it into the new CMS.

What works and what doesn’t

Works:

  1. Lists (Ordered and Unordered but one level only)
  2. Basic formatting (bold, underline, italics)
  3. Horizontal lines
  4. Styles (Headings 1, 2, 3 and Normal Text)
  5. Images
  6. Tables
  7. Hyperlinks
  8. Superscript  
  9. Subscript
  10. Permutation of These Styles

Does not work:

  • Images, inline -- all images will be assumed to be full width and we need to avoid adding images within the text. Icons and others need to be avoided, and will have to be avoided at least from here
  • Tables -- they will not render correctly and tables are evil anyway.
  • Text coloring - because this  does  not  look  good  on multiple devices
  • Text sizes - essentially the same as above, the site needs a very simple and clean look - if you need something big and it’s not a section header, you don’t need something big
  • Text highlighting - because why?
  • Text alignment - Use the styles for the main formatting, everything else will be left/right aligned

On the topic of images

Inline images will not be entered here.

Long images need to be formatted for mobile, in order to avoid have stretched out images.

For now, all images added to documents such as this one will be used as “Framing” images (read: like we have on the blog). So only add images to these documents if their height is 44% of their width.

Will be cut and have a 1/sqrt(5) (44%) aspect ratio.
